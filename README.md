# loragemelo
Gemelo Digital con LoRa
Este repositorio contiene el proyecto elaborado por los alumnos/as del curso *Gemelos Digitales: Transición a la Industria 4.0* impartido en FANOGAR (Jaén) en 2025.

En este proyecto, utilizando un gateway LoRa conectado a una RaspberryPi 5, se conecta a TTN y mediante una integración con Azure IoT Hub, se envían los datos de temperatura y humedad de un sensor LoRa a un Azura Digital Twin de las instalaciones de FANOGAR en Jaén.

El proyecto está estructurado de la siguiente forma:

- Instalación, configuración del Gateway Lora en Raspeberry Pi 5.
- Conexión del Gateway LoRa a TTN.
- Creación y configuración de la aplicación en TTN.
- Integración de TTN con Azure Iot Hub.
- Identidad en Azure con EntraID.
- Administración Azure Iot Hub.
- Interacción con Iot Hub utilizando SDK en Python.
- Administración almacenamiento Azure.
- Almacenamiento de la telemetría de los sensores en una cuenta de almacenamiento de Azure.
- Interacción almacenamiento con SDK en Python.
- Administración CosmosDB.
- Almacenamiento de la telemetría de los sensores en CosmosBD.
- Interacción con CosmosDB utilizando SDK en Python.
- Conexión Iot Hb Azure con CosmosDB con Stream Analitic Job.
- Presenteción de la información con PowerBI.
- Administración de Event Hub Name Space.
- Creación y administracion de Event Hubs.
- Administación de AppFuncion Plan.
- Administración de App Function.
- Desarrollo de Azure Function utilizando el portal.
- Desarrollo de Azure Function utilizando Visual Studio Code.
- Conexión de Event Hub con Azure Function.
- Administración de Azure Digital Twin.
- Azure Digital Twin Explorer. Modelos, Gemelos, 3dScenes.
- Conexión Event Hub con DIgital Twin utilizando una Azure Function.
